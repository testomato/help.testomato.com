The Testomato [REST](https://en.wikipedia.org/wiki/Representational_state_transfer)
API provides simple interfaces for the core Testomato functionality.
To use this REST API, your application makes an HTTP request and parses the
**JSON**-formatted response. All API endpoints require authentication using an **API token**.
